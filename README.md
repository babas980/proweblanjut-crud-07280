---

## Daftar File dan Fungsi

Berikut adalah penjelasan fungsi setiap file dalam direktori proyek:

- **index.php**: Halaman utama atau gerbang awal aplikasi.
- **login.php**: Halaman otentikasi pengguna dengan dukungan fitur Session dan Cookie.
- **register.php**: Halaman pendaftaran akun pengguna baru ke dalam database.
- **dashboard.php**: Halaman pusat informasi yang menampilkan tabel stok barang dan fitur pencarian.
- **connection.php**: Konfigurasi koneksi antara aplikasi PHP dan database MySQL (PDO).
- **create.php**: Halaman formulir untuk menambahkan data produk baru.
- **read.php**: Halaman untuk menampilkan rincian data produk secara spesifik.
- **edit.php**: Halaman formulir untuk memperbarui data produk yang sudah tersimpan.
- **delete.php**: Script untuk memproses penghapusan data produk dari sistem.
- **logout.php**: Script untuk mengakhiri sesi pengguna serta menghapus cookie login.
- **filedatabase.sql**: Berkas script database untuk proses import tabel ke MySQL/phpMyAdmin.
